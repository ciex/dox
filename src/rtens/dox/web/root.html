dox Executable Documentation

Welcome to dox, your friendly neighbourhood browser for executable documentation.

Executable documentation (aka executable specification) lives in code. This means it can be executed and let you know if it contradicts with the production code. Therefore it also doubles as an automated (regression) tests. Three for the price of one!

In other words: documentation that you can trust. A single source of truth.
